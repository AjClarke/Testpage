Ideal Jobs



Our group has a wide selection of different ideal jobs which will lead to varying different career plans. With Alex and Allan wanting to be game developers, Mitch a software engineer, Amit an Application Specialist, Anthony in IT support and Han as a project manager. These role are similar in some aspects as all of these roles will require good communication skills, teamwork and solid general IT knowledge. Some roles will require more specialised knowledge in different areas, with game developers needing more specialised knowledge of programming languages such as Unity, software engineers and applications specialists needing more rounded knowledge to find the right solution for their client or employer, Project managers need to have good leadership skills and IT support need good rounded knowledge in software and hardware to help with troubleshooting. One big thing in common with all of these roles is that in some companies all of these roles are required to build a team for say making a game. Game developer is a more specialised role limiting the potential employment opportunities, while the other roles are more highly sort after which opens up more potential employment opportunities.
